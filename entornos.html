Lista de comandos Git

  • git init - Crea un nuevo repositorio en el directorio actual. (Realizado por Anna Arenas)
  • git clone - Clona un repositorio existente. (Realizado por Dayron Flores)
  • git add - Añade archivos al área de preparación (staging area). (Realizado por Anna Arenas)
  • git commit - Guarda los cambios en el historial del repositorio. (Realizado por Dayron Flores)
  • git push - Envía los cambios al repositorio remoto. (Realizado por Anna Arenas)
  • git pull - Descarga y actualiza el repositorio local con los cambios del remoto. (Realizado por Dayron Flores)
  • git status - Muestra el estado actual del repositorio (archivos modificados, no rastreados, etc.). (Realizado por Anna Arenas)
  • git log - Muestra el historial de commits del repositorio. (Realizado por Dayron Flores)
  • git branch - Lista, crea o elimina ramas en el repositorio. (Realizado por Anna Arenas)
  • git checkout - Cambia de rama o restaura archivos en el directorio de trabajo. (Realizado por Dayron Flores)
  • git merge - Fusiona cambios de una rama con otra. (Realizado por Anna Arenas)
  • git remote - Gestiona conexiones con repositorios remotos. (Realizado por Dayron Flores)
  • git fetch - Descarga cambios del repositorio remoto sin fusionarlos automáticamente. (Realizado por Anna Arenas)
  • git reset - Deshace cambios en el historial de commits. (Realizado por Dayron Flores)
  • git rebase - Reaplica commits sobre otra base, útil para reorganizar un historial. (Realizado por Anna Arenas)
